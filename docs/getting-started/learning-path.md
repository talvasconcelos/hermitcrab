# Learning path

Find the right documentation for what you want to do.

## I just installed HermitCrab

1. Read [Quickstart](quickstart.md) — first conversation in 2 minutes
2. Read [Daily use](../user-guide/daily-use.md) — what the agent can do for you
3. Read [Sessions and memory](../user-guide/sessions-and-memory.md) — how memory works

## I want to talk to HermitCrab from my phone

1. Read [Channels](../user-guide/channels.md) — pick your channel and configure it
2. Read [Gateway operations](../operator-guide/gateway-operations.md) — run the gateway reliably
3. Read [Channel reference](../reference/channel-reference.md) — detailed setup and limits

## I want to schedule recurring tasks

1. Read [Reminders and cron](../user-guide/reminders-and-cron.md) — one-shot reminders and recurring jobs
2. Read [Tools reference](../reference/tools-reference.md) — the `reminder` and `cron` tools

## I'm a developer who wants to extend HermitCrab

1. Read [Architecture](../developer-guide/architecture.md) — how the pieces fit together
2. Read [Agent loop](../developer-guide/agent-loop.md) — the core processing loop
3. Read [Extending tools](../developer-guide/extending-tools.md) — add new tools
4. Read [Contributing](../developer-guide/contributing.md) — how to contribute

## Something is broken

1. Run `hermitcrab doctor` — automated diagnostics
2. Run `hermitcrab status` — runtime and config status
3. Read [Troubleshooting](../user-guide/troubleshooting.md) — common fixes
4. Read [Incident playbook](../operator-guide/incident-playbook.md) — step-by-step recovery

## I want to understand the workspace structure

1. Read [Workspace model](../operator-guide/workspace-model.md) — single and multi-workspace
2. Read [Backups and recovery](../operator-guide/backups-and-recovery.md) — protect your data
3. Read [Config reference](../reference/config-reference.md) — every config field explained

## I want to audit what the agent is doing

1. Run `hermitcrab audit` — recent policy decisions and events
2. Read [Observability and audit](../operator-guide/observability-audit.md) — deep dive
3. Read [Security and permissions](../user-guide/security-and-permissions.md) — safety model

## I want to set up multiple isolated workspaces

1. Read [Workspace model](../operator-guide/workspace-model.md) — multi-workspace architecture
2. Read [Gateway routing](../developer-guide/gateway-routing.md) — how routing decisions work
